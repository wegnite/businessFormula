# 快速开始指南

## 🚀 一键启动

### macOS / Linux 用户
```bash
./start.sh
```

### Windows 用户
双击运行 `start.bat` 文件

## 📋 手动启动步骤

### 1. 确保Python环境
```bash
python --version
# 或
python3 --version
```

### 2. 启动HTTP服务器
```bash
# Python 3.x
python3 -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000
```

### 3. 打开浏览器
访问：`http://localhost:8000`

## 🎯 核心功能

### 📊 总览页面
- 查看总利润、总收入、总成本等核心指标
- 观察收入趋势和成本分析图表
- 数据每30秒自动更新

### 🧮 业务公式页面
- **核心公式**: 总利润 = 总收入 - 总成本
- **一级拆解**: 按渠道维度分解
- **二级拆解**: 按岗位责任源细分
- **成本控制**: 各项成本的详细公式
- **KPI映射**: 岗位与指标的对应关系

### 🗺️ 目标地图页面
- 全国业务分布可视化
- 华东、华北、华南三大区域监控
- 各区域目标完成度显示

### 👥 协同看板页面
- 团队成员与指标绑定
- 实时个人业绩监控
- 团队状态一目了然

### 🎯 OKR关联页面
- 公司-部门-个人三级目标
- 进度可视化展示
- 关键结果详细追踪

## 🎨 界面操作

### 导航切换
- 点击左侧侧边栏切换不同功能页面
- 移动端支持滑动菜单

### 数据交互
- 悬停查看图表详细信息
- 地图支持缩放和拖拽
- 进度条动态加载

### 快捷键
- `Ctrl+E`: 导出数据
- `Ctrl+P`: 打印看板

## 🔧 自定义修改

### 修改业务数据
编辑 `script.js` 文件：
```javascript
// 修改指标基础数据
const metrics = [
    { selector: '.card:nth-child(1) .metric-value', baseValue: 你的数值, variance: 变化范围 },
    // 添加更多指标...
];
```

### 修改样式主题
编辑 `styles.css` 文件：
```css
/* 修改主色调 */
body {
    --primary-color: #你的颜色;
    --secondary-color: #你的颜色;
}
```

### 修改业务公式
编辑 `index.html` 文件中的公式部分，替换为你的业务公式。

## 📱 移动端适配

本项目完全支持移动端访问：
- 响应式布局自动适配
- 移动端菜单按钮

## 🔍 故障排除

### 常见问题
1. **图表不显示**: 确保通过HTTP服务器访问，不要直接打开HTML文件
2. **地图加载失败**: 检查网络连接，地图数据从CDN加载
3. **移动端异常**: 使用较新版本的移动浏览器

### 支持的浏览器
- Chrome 60+
- Firefox 60+
- Safari 12+
- Edge 79+

## 📞 技术支持

如果遇到问题，请：
1. 检查浏览器控制台错误信息
2. 确认Python环境正常
3. 验证网络连接状态
